SOP Structure
=============

.. _sop_index:

This document contains links to the Standard Operating Procedures (SOPs) used to process sequencing data through the Illumina DRAGEN DNA pipeline. These SOPs are categorized into individual steps covering the complete end-to-end workflow. Each SOP contains brief conceptual background, step-by-step commands, outputs, and relevant QC checkpoints.
